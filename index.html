HTML Elements

Block
A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
Inline
An inline element does not start on a new line and only takes up as much width as necessary

Html elements are influnced by type of layout browser engine uses generally Normal Layout a.k.a Flow Layout. Browsers typically display the block-level element with a newline both before and after the element. You can visualize them as a stack of boxes.
